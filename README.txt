READ MEHvordan Roborally spilles og softwaren startes:Man starter med at trykke ”Run” på ”Roborally”. Dernæst åbnes der et vindue, hvor man skal trykke på file, og dernæst new game.Så vælges om spiller 2 – 6 spillere, derefter trykkes OK. Så starter selve spillet. Nu skal hver spiller vælge deres programmeringskort til deres robot. Når de har lagt kortene i den ønskede rækkefølge, trykkes der ”finish programming”. Herefter kan der vælges om alle træk skal udføres på en gang, eller individuelt.Når spillet ønskes at blive afsluttet, så trykkes ”File”, derefter ”Exit” og dernæst ”OK” for at accepterer en nedlukning af spillet. Features: 1. Vi har 8 programmeringskort. 2. Hvis en spiller rykker til et felt, hvor der allerede er en spiller ”skubber” spiller 1 til spiller 2.Set up gameIn order to save and load the game, you have to use local database. To set this up, follow these steps below: 1) start your local server, we can recommend MariaDB2) when the server is running, start your database program, here we can recommend MySQL Workbench3) Here you have to create a database, called pisu4) In this database, you will check if it is running under server statusNext step can be done in two ways, choose the one that feels best for you5) a: in the database, create an account according to the connection class, and allow this user permissions to change in the database.5) b: change the information about the user in the connection class so they match the information in the database, and make sure the user has permission to make changes in the database.If you have followed these steps, you are now ready to play the game, and you will be able to save and load the game, to your local database. Have fun.
